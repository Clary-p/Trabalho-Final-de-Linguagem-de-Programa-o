# Trabalho-Final-de-Linguagem-de-Programação
Programa: Agenda Fofa
Aluna: Clarissa Pietrangelo
Matrícula: 20255551 
Professor: José Eduardo dos Santos Jeremias Junior


  Minha proposta é um programa em C, realizado no CodeBlocks que consiste de uma agenda, onde se tem 5 opções - Adicionar Compromisso, Listar Compromissos, Buscar por Data, Remover Compromisso e Salvar e Sair, fazendo uso de diversas funções, incluindo recursivas, de forma que ao finalizar uma operação, com um “enter”, pode-se retornar para o menu da agenda para visualizar, deletar ou adicionar um compromisso.
  O programa cria um arquivo no computador chamado “Agenda.txt”, de forma que caso o usuário saia do programa, pode executá-lo novamente e os compromissos agendados lá antes vão estar registrados, já que o programa busca-os no arquivo de texto mencionado, o atualizando quando solicitado. 
  Coloquei, também, como escolha estética, um gatinho desenhado com caracteres no topo da agenda, daí vindo o nome, “Agenda Fofa”.
  Cumprem-se os requisitos de utilizar conceitos de arquivo, funções, condicionais, variáveis e muitos outros aprendidos durante o semestre.
